Программа для удаления опенингов и эндингов из аниме.

## Конфиг

В переменную `INPUT_DIR` указывается папка с *сериями.
В переменную `OUTPUT_DIR` указывается папка для сохранения результатов.

\*Для корректной работы у видео должны быть оригинальные аудио дорожки

## Запуск

```bash
python main.py
```
